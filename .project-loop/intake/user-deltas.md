# Пользовательские Дельты

Проект: trackmate
Обновлено: 2026-06-23

Этот файл хранит существенные свежие корректировки, решения и изменения области от пользователя, которые приходят во время шага. Записи компактны и пополняются до отражения в источнике правды и handoff.

## Записи

### Короткое название дельты

ID источника: `S###`

Исходный ввод:

```text
Точная пользовательская дельта или точная выдержка.
```

Нормализация:
- [ ] требование / ограничение / решение:

Маршрутизация:
- [ ] исходный ввод сохранен или записан здесь
- [ ] карта источников обновлена
- [ ] чеклист обновлен
- [ ] план и текущий шаг обновлены
- [ ] реализация направлена в рабочий шаг или отложена
- [ ] reviewer проверил
- [ ] handoff обновлен

Конфликты:
- отсутствуют
