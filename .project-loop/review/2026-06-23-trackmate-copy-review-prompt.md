# Промпт Для Ревью Текстов Trackmate

Скопируй этот файл целиком в нейронку и попроси ее отревьюить тексты. Ниже уже есть контекст продукта, критерии, старые и новые формулировки, а также формат ответа.

```text
Ты редактор интерфейсных текстов для Telegram-бота Trackmate.

Trackmate — бот для маленькой рабочей группы. В нем есть темы:
- Сегодня: одна главная цель-задача дня и вечерний итог
- Прогресс: общая лента закрытых задач и автоматических итогов
- Рутины: ежедневная отметка повторяющихся действий, серии и таблица
- Цели: сезонные цели, недельные обзоры и итог периода

Нужно отревьюить новые пользовательские тексты ниже и предложить улучшения.

Цель ревью:
- сделать тексты лаконичными, понятными и приятными глазу
- сохранить спокойный стиль Trackmate: коротко, по делу, без маркетинга
- убрать английские слова и англицизмы из пользовательского текста
- сделать форматирование ровным для Telegram: заголовки, пустые строки, списки, блоки
- не добавлять лишнюю бюрократию и длинные объяснения
- сохранить текущую механику продукта

Ограничения:
- можно использовать Telegram HTML: <b>, <i>, <blockquote>
- не ломай смысл кнопок и сценариев
- не предлагай новые продуктовые механики, только улучшай текст
- не используй английские термины вроде check-in, review, leaderboard, streak, daily, weekly, outcome, report
- если слово Trackmate встречается как название продукта, его можно оставить
- не делай тексты слишком теплыми, мотивационными или инфантильными
- не добавляй точки в конце коротких кнопок, заголовков и однофразовых toast-ответов, если без них лучше

Формат ответа:
1. Короткий общий вывод: что с текстами хорошо, что режет глаз
2. Список конкретных проблем по приоритету
3. Таблица улучшений:
   - Где
   - Сейчас
   - Предлагаю
   - Почему лучше
4. Финальная версия всех текстов, которые ты предлагаешь поменять
5. Отдельно укажи тексты, которые лучше оставить как есть

Важно: не переписывай все ради переписывания. Если новый текст нормальный, скажи "оставить".
```

## Замены Терминов

| Было | Стало | Комментарий |
| --- | --- | --- |
| check-in | проверка / отметка | В пользовательских текстах не используем check-in |
| leaderboard | таблица | Для рутины сейчас `Рутины: таблица` |
| review | обзор / итог периода | По целям: `Недельный обзор`, `Итог периода` |
| streak / стрик | серия | В таблице рутины: `серия 5 дней` |
| report / отчет | итог | Для задачи дня и прогресса |
| status / статус | состояние | В карточке задачи |
| outcome | результат | В инструкции целей |
| media / медиа | фото или видео | В подсказках ввода |
| alert / алерт | напоминание | В ответах на кнопку |

## Вопросы Для Ревьюера

Проверь особенно:
- не стало ли слишком сухо после удаления точек
- звучит ли `Рутины: таблица` понятно или лучше другое короткое название
- нормальна ли кнопка `🏁 Подвести итог`
- лучше ли `цель-задача дня` или стоит выбрать один термин
- не перегружен ли шаблон целей
- достаточно ли понятно, что рутина спрашивается одной карточкой в день
- хорошо ли звучит `Что помогло? Что мешало? Что изменишь завтра?`
- не звучат ли nudges про сезонные цели слишком назидательно
- нет ли мест, где нужно вернуть более человеческую интонацию

## Что На Выходе Нужно От Модели

Нужен не общий текст, а готовый список правок. Если предлагаешь заменить текст, дай полную финальную версию сообщения целиком, включая emoji, HTML-теги и переносы строк.

Пример формата:

| Где | Сейчас | Предлагаю | Почему |
| --- | --- | --- | --- |
| Рутины, таблица | `🏆 <b>Рутины: таблица</b>` | `🏆 <b>Рутины: результаты</b>` | ... |

После таблицы дай финальный блок:

```text
Финальные тексты для замены:

1. TodayControlText
...
```
